# Netflix ETL and Visualization Project

This project is an end-to-end data pipeline and analytics solution built with **Python**, **PostgreSQL 16 (Docker)**, and **Power BI**.

## Overview

- Extract and clean the Netflix titles dataset using Python (pandas, SQLAlchemy).
- Load the processed data into a PostgreSQL 16 database running in a Docker container.
- Build an interactive Power BI dashboard to analyze total titles, movies vs TV shows, genres, ratings, and release trends connected directly to PostgreSQL.

## Tech Stack

- Python (pandas, SQLAlchemy)
- PostgreSQL 16 (Docker)
- Power BI
